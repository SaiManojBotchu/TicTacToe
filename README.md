## Getting Started 🚀

**1. [ Setup Flutter](https://flutter.io/setup/)**

**2. Clone the repo**

```sh
$ git clone https://github.com/SaiManojBotchu/TicTacToe.git
$ cd TicTacToe/
```

**3. Install dependencies**
```sh
$ flutter pub get
```

**4. Run app**
```sh
$ flutter run
```
